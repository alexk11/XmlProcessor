
XmlProcessor v1.0 

Usage: java -jar XmlProcessorDrv [-d <Dir>] [-h] [-v] -elementCount <name>

 -h,--help   Display this help
 -v          Version of the program

 -d <Dir>    Folder with XML files to be validated
 -elementCount  Finds the count of element by node name in XML 
  
 Example:  java -jar xmlProcessor.jar --help -v -d "C:\\XmlSample" -elementCount "Street"
